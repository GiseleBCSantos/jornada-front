Explicação dos valores de display:

  • block: Ocupa toda a largura disponível e inicia em uma nova linha (ex: div, p, h1).
  • inline: Ocupa apenas o espaço necessário e não quebra a linha (ex: span, a, strong).
  • inline-block: Comporta-se como inline, mas permite definir altura e largura.
  • flex: Cria um layout flexível, distribuindo elementos dinamicamente dentro de um contêiner.
  • grid: Define um layout baseado em grade, organizando os elementos em linhas e colunas.
  • none: Oculta o elemento completamente, sem ocupar espaço no layout.
